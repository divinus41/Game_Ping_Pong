# Ping Pong

## Description

Play tennis against yourself with two rackets and try to keep the ball in play as 
long as possible.

## Control

Move the 'MOUSE' to the 'LEFT' to move the racket to the left! Move the 'MOUSE' to 
the 'RIGHT' to move the racket to the right!

## Features

* Simple gameplay
* Only mouse control
* 60 FPS
